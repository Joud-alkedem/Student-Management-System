# Student Management System V2

A Java console application for managing student records using Object-Oriented Programming (OOP).

## Features

- Add Student
- Delete Student
- Edit Student
- Search Student by ID
- Display All Students
- Sort Students by GPA
- Find Highest GPA
- Find Lowest GPA
- Save Data to File
- Load Data Automatically

## Technologies

- Java
- OOP
- ArrayList
- File I/O
- Collections Framework
